# Project Brief

## Product

DreamVoice AI is a bedtime story web app for parents. A parent creates a child profile, chooses a story theme, and receives a personalized bedtime story with optional narration.

The emotional goal is parent-child bonding through personalized stories. Phase 1 should prove the core loop without paid voice cloning or subscription complexity.

## Target User

Primary user: a parent or guardian who wants quick, personalized bedtime stories for a child.

Secondary user: the child who listens to the story or reads along with the parent.

## Phase 1 MVP

The MVP includes:

- Parent registration and login.
- Parent profile.
- Child profile.
- Six story theme buttons.
- Automatic prompt generation.
- AI story generation.
- Google TTS narration using free-tier-compatible voices.
- Optional manual parent recording.
- Audio/story library.
- Basic usage tracking for free-tier limits.

## Story Theme Buttons

Use six preset choices:

1. Magical Adventure
2. Calm Bedtime
3. Brave Little Hero
4. Kindness and Sharing
5. Animal Friends
6. Learning Quest

When a parent selects a button, the app builds the prompt from:

- Child name
- Child age
- Child interests
- Preferred language
- Bedtime tone
- Selected theme

## Phase 1 Non-Goals

Do not build these yet:

- Voice cloning.
- Subscription billing.
- Family plan.
- School or enterprise plan.
- Mobile app.
- Background music.
- Multi-voice narration.
- Advanced admin analytics.
- Public story sharing.

## Success Criteria

The MVP is successful when a parent can:

1. Sign up and log in.
2. Create a child profile.
3. Pick one of six story buttons.
4. Generate a personalized story.
5. Generate TTS narration if quota allows.
6. Optionally record their own narration.
7. Return later and play/read saved stories.
